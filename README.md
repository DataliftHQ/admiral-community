## ğŸ’¼ About Admiral

Admiral eliminates infrastructure bottlenecks by giving application teams self-service access to platform-approved infrastructure while maintaining security and compliance.

**What Admiral Does:**

- **Self-Service Infrastructure:** Deploy Redis, RDS, S3, and more in minutes from a curated Terraform module catalog
- **Unified Deployment:** Deploy Kubernetes workloads across any cloud or on-premises with one system
- **Platform Guardrails:** Maintain control through policies, approvals, and audit trails

**Who Uses Admiral:**

- Platform engineering teams enabling developer autonomy
- DevOps teams managing multi-cloud deployments
- Engineering organizations scaling their infrastructure workflows
